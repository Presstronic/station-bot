# Station Bot v0.3.3-beta

## Release Date

2026-05-30

## Summary

`v0.3.3-beta` is a substantial feature and platform release for Station Bot. It completes the guild-config database migration effort, adds automated Discord event reminders, introduces the new `/station-timer` and `/exec-hangar` command groups, and closes several workflow and reliability gaps in nominations, manufacturing, verification, and operator visibility.

## Main Changes

- Completed the move of guild-specific Discord configuration from `.env` into database-backed guild configuration, including startup seeding and multi-guild feature wiring.
- Added automatic event reminders for Discord scheduled events, including reminder scheduling, cleanup, and per-guild configuration support.
- Added `/station-timer` for start, stop, and list timer workflows with persisted timers, DM expiry notifications, and optional voice-text follow-up notifications.
- Added `/exec-hangar` with local timer derivation, startup sync from `exec.xyxyll.com`, admin re-sync/manual sync, and persisted global state.
- Improved member and operator visibility with `/healthcheck` version output, pending nominations in `/my-nominations`, and corrected nomination digest behavior when there is nothing to send.
- Fixed workflow gaps in manufacturing requester notifications and RSI nickname canonicalization.
