# Station Bot v0.3.3-beta.1

## Release Date

2026-05-30

## Summary

`v0.3.3-beta.1` is a release-process correction for `v0.3.3-beta`. It preserves the same application feature set while fixing the release workflow so tagged releases stop after validation, GHCR image publish, and GitHub Release creation, leaving production deployment fully manual.

## Main Changes

- Removed automatic production deployment from the tag-triggered release workflow.
- Documented the required manual production deployment flow, including pre-deploy database backup, smoke testing, and rollback.
- Added a manual deployment and rollback runbook to support safer production releases while auto-deploy remains disabled.

## Issues Addressed

- No new product issue was addressed in this patch release. This release exists to correct the release workflow and operational documentation for the `v0.3.3-beta` release line.

## Migration / Rollout Notes

- This release contains no new database migrations.
- Application feature scope is unchanged from `v0.3.3-beta`.
- Production deployment for this release should follow [manual-deployment-checklist.md](./manual-deployment-checklist.md).
