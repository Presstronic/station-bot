# Member Verification

Verify your RSI identity to unlock full org member access.

---

## Steps

**1. Run the command**
```
/verify in-game-name: YourRSIHandle
```
Enter your RSI handle exactly as it appears on your profile page.

**2. Add the code to your RSI bio**
The bot will reply with a short code that looks like:
```
DRDNT-a1b2c3d4e5f6g7h8
```
Go to [robertsspaceindustries.com](https://robertsspaceindustries.com) → your profile → **Edit** → paste the code into your **short bio**. Save.

**3. Click Verify**
Return to Discord and click the **Verify** button in the bot's reply. The bot checks your bio live — if the code is there, you're in.

**4. Clean up**
Once verified, remove the code from your RSI bio. It's no longer needed.

---

## Notes

- The bot's reply is only visible to you
- Your server nickname will be set to your RSI handle automatically
- If you see a "session has expired" message, just run `/verify` again
- If your nickname couldn't be updated, contact a server administrator

---

## Troubleshooting

| Problem | Fix |
|---|---|
| "Could not verify citizenship" | Make sure the code is saved in your **short bio**, not your biography |
| "Please enter your RSI handle only (e.g. Testhandle123), not a full URL." | Enter your handle only — not a full URL |
| Session expired | Run `/verify` again and complete within the session window |
